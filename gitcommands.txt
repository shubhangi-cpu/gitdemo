git --version: It shows you the version of the git software
$ git init : It intialized the git so that a specific directory is manged by the git 
program, It starts tracking the changes in the specifed directory.

$ git status ->it display status of file you have changed ,removed or added into the git directory

$ git add file1 ->for adding file into the git demo
$ git add file1 file2
$ git add *
$ git  add -A
git rm --cached file1 or git rm --cached *: It will remove the file or folders from the 
caching /staging area to the working area
git rm --cached * -f: In case files are not moving back to the working area from staging area we
 have to use the force option
 git commit : it will commit all the files from staging area to the local repository
 git commit -m "updated version for git commands"
 git log: will show you the commit log for the commited files int the local repositary
 git diff: it will show the diff bet the current file and the latest commited file
git show: will show the diff bet the latest commit and current file
